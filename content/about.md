About
==========================

This webb page is built with Pico CMS to manage markdown files that contain the content of the pages. Pico is a flat-file CMS which means that all content/data is stored on files instead of a server.

For the html structure we use Twig to output variables in html. With this we can split parts of our pages into files. (For example header.twig, nav.twig and footer.twig)

The styling of the webb site is done with Sass. Sass is a CSS pre-processor that adds the ability to use variables, modules and more. 

I use two fonts that i import from Google Fonts, Montserrat for headings and Work Sans for the paragraph text. I will probably change fonts in the future because i think they look too similar.
